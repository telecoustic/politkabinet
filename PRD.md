# Политкабинет — PRD (Product Requirements Document)

## Обзор продукта

Фановый публичный сервис, где любой человек проходит ситуационный тест политических взглядов, получает свой политический тип с описанием и может сравнить результат с друзьями.

## Проблема

Люди интересуются политическими взглядами — своими и чужими — но не хотят спорить о политике напрямую. Нет простого способа понять, "кто ты" и "насколько мы с другом похожи" без конфликта.

## Решение

Тест из бытовых ситуаций (не прямых политических вопросов), который незаметно измеряет 4 оси взглядов и выдаёт ироничный тип-ярлык. Механика сравнения с друзьями через ссылку создаёт виральный эффект.

## Происхождение идеи

Реальный кейс: компания друзей посмотрела рилс про политические взгляды, захотели понять, кто есть кто. Прошли тест вместе, обсудили — оказалось больше общего, чем различий. Провели время полезно и интересно.

## Целевая аудитория

- Люди 20-40 лет, которым кинули ссылку в чат
- Компании друзей, которые хотят развлечься с пользой
- Люди, которым интересно разобраться в своих взглядах без политической нагрузки

## Модель: 4 оси (на базе 8values)

| Ось | 0 | 100 |
|---|---|---|
| Экономика | Равенство | Рынок |
| Дипломатия | Глобализм | Нация |
| Государство | Свобода | Авторитаризм |
| Общество | Прогресс | Традиция |

16 типов-ярлыков (все комбинации полюсов). Каждый тип — ироничное название + описание.

## Пользовательский путь (Итерация 1)

### Вход в тест (общий для всех сценариев)

1. Заходит на politkabinet.ru (напрямую или по ссылке друга)
2. Экран приветствия: заголовок + подводка
3. Поле "Сколько тебе полных лет?" → поле "Как тебя зовут?"
4. Кнопка "Поехали" → начинается тест

### Прохождение теста

24 ситуационных вопроса, по одному на экран. Прогресс-бар сверху.

### Экран результата

1. Тип-ярлык крупно ("Корпоративный монарх")
2. Ироничное описание типа (заготовленное, подробное)
3. 4 шкалы с визуализацией
4. Код результата (85-41-81-55)
5. Кнопка "Поделиться" → копирует ссылку `politkabinet.ru/t/85-41-81-55`
6. **Блок "Сравни с другом":**
   - Поле "Код друга" — заполнено автоматически (если зашёл по ссылке) или пустое (ручной ввод)
   - Кнопка "Сравнить" → показывает блок сравнения ниже
   - Кнопка "×" — очистить поле, скрыть сравнение
   - Можно менять код и сравнивать с разными друзьями по очереди

### Блок сравнения (появляется при наличии кода друга)

- 4 шкалы рядом: "Ты" и "Друг" (визуально)
- По каждой оси: текстовый комментарий о разнице
- Общий вывод: "совпадаете на X%, главное расхождение — ось Y"
- Всё вычисляется на клиенте, заготовленные шаблоны текстов

### Сценарий: Групповой

Каждый в чате проходит и кидает свою ссылку. Любой может вставить чужой код в поле и сравнить.

## Экран результата — детали

### Описание типа (заготовленное)

Для каждого из 16 типов:
- Название + emoji
- Краткое ироничное описание (2-3 предложения)
- Развёрнутое описание (1-2 абзаца): как этот тип смотрит на мир, что ценит, с кем чаще всего спорит, исторические аналогии

### Сравнение двух профилей

- 4 шкалы рядом (визуально)
- По каждой оси: текстовый комментарий о разнице
- Общий вывод: "совпадаете на X%, главное расхождение — ось Y"
- Всё заготовленное, без API

## Технические решения (Итерация 1)

- **Без бэкенда** — всё на клиенте
- Код результата (85-41-81-55) передаётся в URL
- Сравнение вычисляется в браузере
- Описания типов — захардкожены в коде
- Тексты сравнений — шаблоны с подстановкой значений
- Стек: статичный сайт (HTML/CSS/JS или Next.js static export)
- Деплой: politkabinet.ru на VPS (72.56.108.79)

## Итерация 2 (будущее, платная)

- Раунд 2: 16 конкретных вопросов, уточняющих позиции Раунда 1
- Сравнение "ты vs ты": сдвиг между раундами с объяснением
- AI-разбор (Claude API): персональный анализ по конкретным ответам
- Оплата за AI-разбор (сумма TBD, зависит от стоимости запроса)

## Метрики успеха

- Конверсия: % зашедших → прошедших тест до конца
- Виральность: % прошедших → поделившихся ссылкой
- Возврат: % друзей, перешедших по ссылке → прошедших тест
