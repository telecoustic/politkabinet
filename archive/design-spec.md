# Political Quiz — Design Spec

## Summary

Ситуационный тест политических взглядов по 4 осям (модель 8values). Ироничная подача, без прямых политических вопросов. Один статический HTML-файл, mobile-first, деплой на aristokrab.ru/quiz.

## Цель

Пройти втроём на телефонах, обсудить результаты на большом экране. Развлечение + реальное понимание своих взглядов.

## 4 оси

| Ось | Полюс 0 | Полюс 100 |
|---|---|---|
| Экономика | Равенство | Рынок |
| Дипломатия | Глобализм | Нация |
| Государство | Свобода | Авторитаризм |
| Общество | Прогресс | Традиция |

## Вопросы

- 24 вопроса, по 6 на ось
- Формат: бытовая/житейская ситуация + 4 варианта ответа (конкретные действия)
- Каждый вопрос затрагивает 1-2 оси
- Пользователь не понимает, что именно измеряется
- Все 4 варианта должны быть равно привлекательными (никаких "очевидно плохих" ответов)

## Подсчёт

- Стартовая позиция: 50 по каждой оси
- Каждый вариант сдвигает на +/- 5-10 баллов по затронутым осям
- Итог: 4 числа 0-100
- Зона центриста: 40-60 — ось не вдавливается в полюс

## Типы-ярлыки

16 базовых типов (2^4 комбинаций полюсов) + промежуточные варианты при попадании в зону 40-60.

Каждый тип: ироничное название + emoji + описание в 2-3 предложения.

Примеры базовых типов:
- Рынок + Нация + Авторитаризм + Традиция = **"Железный канцлер"** — "Порядок, скрепы, частная собственность. Бисмарк одобряет."
- Равенство + Глобализм + Свобода + Прогресс = **"Космический хиппи"** — "Границы — выдумка, собственность — иллюзия, будущее — за коммунами на Марсе."
- Рынок + Глобализм + Свобода + Прогресс = **"Кремниевый евангелист"** — "Uber для всего, блокчейн для остального, государство — помеха."
- Равенство + Нация + Авторитаризм + Традиция = **"Народный трибун"** — "Всё наше, чужим не дадим, вождь знает лучше."

Остальные 12 типов прорабатываются при реализации.

## UX-флоу

Один HTML-файл, mobile-first.

### Экран 1: Приветствие
- Название теста (ироничное)
- Подводка в 1-2 предложения
- Поле "Как тебя зовут?"
- Кнопка "Поехали"

### Экран 2: Вопросы (x24)
- Один вопрос на экран
- Ситуация — текст
- 4 варианта — крупные карточки (тап-френдли)
- Прогресс-бар сверху
- Тап по варианту = переход к следующему

### Экран 3: Результат
- Тип-ярлык крупно + emoji
- Ироничное описание (2-3 предложения)
- 4 шкалы с подписями полюсов
- Код результата для копирования/диктовки

### Код результата
- Формат: `73-45-82-31` (4 числа через дефис, по одному на ось: экономика-дипломатия-государство-общество)
- Имя не включается
- Легко продиктовать голосом (11 символов макс)
- Сравнение: Руслан собирает 3 кода, отдаёт Claude для живого разбора

## Визуальный стиль

**"Ироничный политкабинет":**
- Фон: приглушённый бежевый/газетный
- Заголовки: крупный сериф (Georgia), как газетные шапки
- Текст: sans-serif для читаемости на мобильном
- Карточки ответов: лёгкая тень, крупные (палец не промахнётся)
- Акцентные цвета осей на экране результата
- Emoji вместо картинок (серп и молот, статуя свободы, корона, ракета и т.д.)
- Тип-ярлык: крупно по центру, как "приговор"

## Технический стек

- Один файл index.html (HTML + CSS + JS inline)
- Vanilla JS, никаких фреймворков
- CSS: flexbox, media queries для мобильных
- Шрифты: системные (Georgia + system-ui)
- Никакого бэкенда, всё в браузере

## Деплой

1. `scp index.html ruslan@72.56.108.79:~/quiz/index.html`
2. Добавить location в nginx:
   ```
   location /quiz {
       alias /home/ruslan/quiz;
       index index.html;
   }
   ```
3. `sudo nginx -t && sudo systemctl reload nginx`

URL: `https://aristokrab.ru/quiz`
