## Наблюдения по методике

1. **Абстрактные vs конкретные вопросы дают разные результаты.** Особенно по оси "Государство": люди на абстракциях за свободу, на конкретике — за контроль. R2 был нужен.

2. **Вопросы без контекста провоцируют рассуждения, а не ответы.** Пример: "друг потерял работу" — непонятно почему, невозможно выбрать. Контекст ("сократили при массовом урезании штата") решает.

3. **Все 4 варианта должны быть равно привлекательны.** Фраза "нечего скрывать — нечего бояться" — маркер, а не вариант. Люди распознают ловушку.

4. **Синхронное прохождение с обсуждением** — ценно для группы. Фиксированный порядок вопросов обязателен для этого режима.
