Напиши пост для соцсетей о Политкабинете.

$ARGUMENTS — платформа и тема. Примеры:
- `telegram` — пост для Telegram-канала
- `instagram` — пост для Instagram Stories/Reels
- `telegram запуск` — пост-анонс запуска
- `telegram результаты` — пост с интересной статистикой

Контекст продукта:
- Политкабинет (politkabinet.ru) — бесплатный тест политических взглядов
- 24 бытовые ситуации, 4 оси, 16 типов-ярлыков
- Фишка: сравнение с друзьями через ссылку
- Тон: ироничный, умный, с подмигиванием. Не академический, не агрессивный

Требования к посту:
- Без эмодзи (если не попросят явно)
- Короткий (Telegram: до 500 символов, Instagram: до 300)
- Заканчивается ссылкой politkabinet.ru
- Провоцирует кликнуть и пройти тест
- Провоцирует переслать друзьям

Если тема "результаты" — сначала запроси данные через Метрику API (счётчик 108526234, токен из memory) и используй реальные цифры.
