# Политкабинет

Ситуационный тест политических взглядов по 4 осям. Создан 4 апреля 2026.

## Что внутри

| Файл | Что это |
|---|---|
| `round1.html` | Раунд 1: 24 ситуационных вопроса (бытовые, замаскированные) |
| `round2.html` | Раунд 2: 16 конкретных вопросов (уточняет позиции R1) |
| `results.md` | Результаты первого тестирования (Руслан, Юля, Ия) + анализ |
| `design-spec.md` | Дизайн-спецификация теста |
| `implementation-plan.md` | План реализации |

## Как это работает

### Модель: 4 оси (на базе 8values)

| Ось | 0 | 100 |
|---|---|---|
| Экономика | Равенство | Рынок |
| Дипломатия | Глобализм | Нация |
| Государство | Свобода | Авторитаризм |
| Общество | Прогресс | Традиция |

### Механика

- Каждый вопрос — бытовая ситуация + 4 варианта ответа (конкретные действия)
- Пользователь не понимает, какую ось измеряет вопрос
- Каждый вариант сдвигает 1-2 оси на +/- 3-10 баллов
- Стартовая позиция: 50 по каждой оси (R1) или значения R1 (R2)
- Зона 40-60 = центрист, не вдавливается в полюс
- Результат: тип-ярлык (16 базовых) + 4 шкалы + код

### Код результата

Формат: `75-33-75-47` (экономика-дипломатия-государство-общество, каждое 0-100).

### Два раунда

- **R1 (24 вопроса):** бытовые, ситуационные, абстрактные. Выявляет базовый профиль.
- **R2 (16 вопросов):** конкретные, приземлённые. Стартует от R1, уточняет. Показывает сдвиг.

R2 нужен потому что люди на абстракциях отвечают "за свободу", а на конкретике ("штраф за 62 в зоне 60") — "правила есть правила".

## Идеи для развития

- Раунд 3: дилеммы без "правильного" ответа (trolley problem стиль)
- Сравнение профилей на одном экране (для группового обсуждения)
- Историческое сопоставление: "ваш профиль похож на..." (реальные политические фигуры)
- Адаптивные вопросы: R2 подбирается по слабым местам R1
- Экспорт в PDF / картинку для шаринга
