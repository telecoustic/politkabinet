# Политкабинет

Фановый публичный сервис — ситуационный тест политических взглядов по 4 осям (модель 8values). 24 бытовые ситуации, 16 типов-ярлыков, сравнение с друзьями через ссылку.

Домен: politkabinet.ru (планируется)
Текущий деплой: https://aristokrab.ru/quiz/

## Структура проекта

| Файл/папка | Что это |
|---|---|
| `app/` | Next.js + shadcn/ui + Tailwind — основное приложение (лендинг) |
| `PRD.md` | Product Requirements Document — описание продукта, флоу, итерации |
| `ajtbd-analysis.md` | AJTBD-анализ по Замесину: 5 сегментов, граф работ, текст лендинга |
| `landing-copy-final.md` | Финальный копирайт лендинга (AJTBD-структура + ирония) |
| `types-descriptions.md` | Описания всех 16 типов + шаблоны сравнения + анализ вопросов |
| `landing.html` | HTML-прототип лендинга (первая версия) |
| `round1.html` | Раунд 1: 24 ситуационных вопроса (legacy, статичный HTML) |
| `round2.html` | Раунд 2: 16 конкретных вопросов (legacy, статичный HTML) |
| `design-spec.md` | Оригинальная дизайн-спецификация теста |
| `implementation-plan.md` | Старый план реализации (до переезда на Next.js) |
| `results.md` | Результаты первого тестирования (Руслан, Юля, Ия) |

## Модель: 4 оси

| Ось | 0 | 100 |
|---|---|---|
| Экономика | Равенство | Рынок |
| Дипломатия | Глобализм | Нация |
| Государство | Свобода | Авторитаризм |
| Общество | Прогресс | Традиция |
